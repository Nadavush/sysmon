## Abstract Idea

Supposed to be a command-line tool that tracks CPU, memory, network, and disk usage in real time and displays live metrics of them in the terminal with a clean, colored interface. The tool additionally logs historical data to a file, which can be formated for both CSV and JSON for its output.
